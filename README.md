16:00 - 17:00 [ Dragon Boat Festival with RythConnection](https://sjpl.bibliocommons.com/events/67f6e5632b0fb15d9faef3da)

Come to Calabazas Library to celebrate the Dragon Boat Festival (端午節). This festival is a vibrant celebration of cultural heritage and traditions.
